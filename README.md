# TMHa

### 開発環境
- Xcode6
- Objective-C

### 概要
- 2014年Tokyo metroコンテストのため作ったアプリ
- 電車でスマホで乗り忘れることを防止するためのアプリ

### このアプリについて
- メトロAPIを利用してメトロ線の駅と駅の間の所用時間が計れる
- 所用時間でタイマー設定ができる
- タイマー設定の時アラーム音の選択ができる
- よく使う駅の設定やアラーム設定などの保存機能がある
- オフラインでも使える
